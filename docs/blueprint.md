# **App Name**: MY AI News

## Core Features:

- Content Fetching: Fetch the latest news posts from Firestore with fields for title, category, imageUrl, date, and content.
- Header Component: Implement a header with a white background, the MY AI logo text, a dark mode toggle, and a hamburger menu for mobile navigation.
- News Ticker: Create a scrolling news ticker that displays the latest post titles and dates with a pause button.
- Hero Card Carousel: Design a large, centered card that displays a high-quality image, category tag, title, and a 'Continue Reading' link, with left/right arrow buttons for a carousel effect.
- Dynamic Category Tags: Generate and display dynamic category tags based on the categories of content from the Firebase.
- Dark Mode Toggle: Implement a dark mode toggle that dynamically switches between light and dark color schemes.
- Image Generation: Tool for creating blog post images based on article content using a generative AI model.

## Style Guidelines:

- Primary color: Deep Blue (#3B82F6) for a professional and trustworthy feel.
- Background color: Light Gray (#F9FAFB), a subtle and clean backdrop, providing contrast while remaining unobtrusive.
- Accent color: Light Blue (#60A5FA), to create visual interest in the news ticker.
- Headline font: 'Poppins' (sans-serif) for bold titles and 'Inter' (sans-serif) for body text, which offers a contemporary and precise readable feel.
- Use Lucide-React icons for a clean, modern look. Moon icon for dark mode, hamburger menu for mobile navigation, arrow icons for carousel.
- Mobile-first, responsive design to ensure readability on all devices. Centered card layout for the hero section. Clean and intuitive navigation.
- Subtle animations for the news ticker and dark mode toggle to enhance user experience.
